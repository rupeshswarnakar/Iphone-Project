# iPhone Sales Data Pipeline Project

## Overview
This project implements a Bronze-Silver-Gold data pipeline using PySpark and Hive.

## Architecture
- Bronze Layer: Raw CSV ingestion
- Silver Layer: Data cleaning and transformations
- Gold Layer: Star schema (fact + dimension tables)

## Technologies Used
- PySpark
- Hive
- HDFS

## Features
- Data partitioning by date
- Data type transformations
- Fact and dimension modeling
